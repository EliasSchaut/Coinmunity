# Coinmunity

A community balance management platform for shared resources and expenses.

## Features

### User Management
- User registration with name, email, and personal balance
- Role-based access control (Users and Administrators)
- Authentication via Logto integration

### Balance & Transactions
- Personal balance tracking for each user
- Purchase items (drinks, snacks, etc.) from inventory
- Peer-to-peer transactions
- Recurring community contribution deductions
- Balance top-up system (manual entry by admins or via payment APIs)

### Administration
- Administrative dashboard for platform management
- Manual balance adjustments and transaction oversight
- Payment gateway integration support
- User and inventory management

### API Access
- RESTful API for programmatic access
- API key authentication for secure operations
- Comprehensive endpoints for all core features
